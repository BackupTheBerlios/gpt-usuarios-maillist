Bem, em Setembro houve muita comunicação entre interessados no GPT.

Pelo que percebi a linguagem é voltada ao aprendizado de iniciantes (um
deles escreve esta mensagem).

A instalação do ambiente para GPT já é difícil – as instruções no arquivo "
INSTALAÇÃO.txt" são para quem já sabe programar por que eu não consegui
criar o ambiente para o GPT – no entanto, o problema mais urgente é um
sistema melhor de comunicação.

Sem comunicação entre os interessados o GPT não chegará ali na esquina. Uma
excelente opção é usar um sistema de fóruns avançado e de simplerrima
administração, como o phpBB (livre) http://pt.wikipedia.org/wiki/PhpBB Vejam
como ele é http://www.suportephpbb.org/forum/ - o próprio suporte do projeto
no Brasil usa o phpBB como sistema de fóruns.

Apostemos num sistema deste para conversar sobre o GPT, pois o atual é
difícil e inconveniente de usar. Acaba desestimulando mais ainda os novatos
em programação a conhecer o GPT, estudá-lo e ajudar a desenvolvê-lo.

Essa é uma ótima solução para alavancar o GPT; torná-lo conhecido com um
fórum adequado.
