Oi.

Eu gostaria de sugestões sobre como gerar código para acessar parâmetros e
variáveis locais de procedimentos e funções.

Segue abaixo um exemplo em gpt e o equivalente em CI:

// Exemplo em g-portugol

algoritmo exemplo;
início
    soma( 2, 7 );
fim
procedimento soma( x: inteiro, y: inteiro )
início
    imprima( "A soma é: ", x + y );
fim



// Exemplo em CI

// soma( 2, 7 );
param 2
param 7
pcall soma, 2

stop
proc soma
 t1 := __SP + 0 // primeira variável
 t1 := *t1
 t2 := __SP + 4 // segunda variável
 t2 := *t2

 // imprima( "A soma é: ", x + y );
 param "A soma é: "
 t3 := t1 + t2
 param t3
 pcall imprima, 2

end-proc

Neste exemplo, o compilador geraria variáveis temporárias que teriam o valor
dos parâmetros (ou variáveis locais, tanto faz). __SP seria um registrador
interno da VM que teria a posição da nossa pilha.

Apesar de ser válido o código acima (se esqueci de algo, desculpem-me), não
ficou muito bonito. Mas se pesquisarem vão ver que alguns compiladores geram
um código similar a esse para o acesso a dados locais.

Vocês teriam alguma sugestão ???
