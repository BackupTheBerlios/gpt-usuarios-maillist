Oi pessoal.

Na teoria, com as instruções para manipulação de ponteiros, a gente consegue
traduzir qualquer código em gpt que use vetores ou structs para CI.

Porém, só com as instruções para manipulação de vetores, a gente não
consegue traduzir o uso de structs para CI.

Resumindo, só com as instruções para manipulação de ponteiros a gente
consegue fazer o que precisar (manipular structs e vetores).

Com isso, a pergunta é: precisamos manter as instruções para manipulação de
vetores ???
