Na idéia mais simples (sem alocação dinâmica de matrizes) a atribuição
poderia ser feita entre 2 matrizes que tivessem o mesmo número de elementos,
mas não necessariamente a mesma dimensão.



> E o seguinte, seria válido?
>
> m1 : matriz[1][2] de inteiros;
> m2 : matriz[1][2] de inteiros;
>
> m1 := m2;


Sim, seria.

E se tivessemos:
m3: matriz[2][1] de inteiros;
m4: matriz[2] de inteiros;

m1 := m3 // válido
m4 := m1[0] // válido
m1 := m4 // inválido, número de elementos diferentes
