Eu comecei a programar +- em 1983. De lá para cá eu aprendi algumas

clipper, C++, java (pouca coisa), haskell (pouca coisa), prolog, shell

desenvolvimento) como Delphi, C++ Builder, Clarion, ... No final de tudo
isso, eu sempre tenho a mesma impressão: só em C/C++/Assembly eu realmente
vejo o que está acontecendo e consigo fazer o que eu quero "da melhor forma
que eu sei".

Eu acho legal criar coisas novas, e não usar coisas prontas. Se eu tivesse
capacidade para isso eu provavelmente implementaria todas as minhas
bibliotecas de código, "just for fun"...

utilizar, sempre que possível, C/C++.

De uns 4 anos para cá eu só aprendo novas linguagens por algum interesse

pesquisas na área de linguagens, compiladores, parsers e máquinas virtuais).
Agora, aprender uma nova linguagem para fins profissionais, para mim, não
gera mais tanto interesse. É só começar a avaliar algumas linguagens que
nota-se claramente que dá pra se fazer o que quer com qualquer linguagem,
algumas de forma mais fácil, e outras de forma mais difícil, mas sempre se
consegue fazer o que quer.

Novos conhecimento sempre são válidos, claro, mas eu ainda tenho tanta coisa
pra aprender no pouco que tento me especializar que aprender mais uma
linguagem não me anima mais como antes hehehe. Tô ficando velho mesmo
hahaha.
