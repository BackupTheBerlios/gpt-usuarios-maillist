Para o slackware, utilize a versão 1.0 disponível na página do
projeto, pois é mais complicado fazer o utf8 funcionar legal nele.

A proposito, ontem coloquei um novo pacote, com correções, em:

http://sourcecraft.info/mystuff/gpt/gpt-1.0-utf.tar.gz

Eis o processo que acabei de fazer no Ubuntu (livecd), com o novo pacote:

$ sudo aptitude install g++ make antlr libantlr-dev libpcre3-dev nasm

$ wget http://sourcecraft.info/mystuff/gpt/gpt-1.0-utf.tar.gz
$ tar xvf gpt-1.0-utf.tar.gz
$ cd gpt-1.0
$ ./configure
$ make
$ sudo make install

$ gpt -i exemplos/olamundo.gpt
