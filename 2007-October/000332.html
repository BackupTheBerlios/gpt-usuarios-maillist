Acabei de criar uma nova lista para commits no repositório: gpt-commit

Detalhes:
https://developer.berlios.de/mail/?group_id=5791

Para todos os commits realizados no repositório, um email será enviado
para esta lista automaticamente, com as informações de commit
(revisão, autor, arquivos afetados e o diff). Então, é uma lista
"read-only", digamos assim. Já fiz alguns testes e está funcionando.

Se acharem útil, está aí.
