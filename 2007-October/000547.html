Agora a pouco eu vi esse código no wiki:

  se tipo( a ) == literal então
      imprima( "é literal", a );
  senão se tipo( a ) == inteiro então
      imprima( "é inteiro", a );
  senão
      imprima( "é de outro tipo", a );
  fim-se

Pergunta: vamos permitir essa....."consecutividade" para a estrutura
"se"? Pela gramática, isso não é possível. Segundo ela, deveria ser:


  se tipo( a ) == literal então
      imprima( "é literal", a );
  senão
    se tipo( a ) == inteiro então
      imprima( "é inteiro", a );
    senão
      imprima( "é de outro tipo", a );
    fim-se
  fim-se
