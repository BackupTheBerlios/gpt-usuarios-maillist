Olá,

o gpt foi originalmente escrito com codificação Latin1, só um tempo depois
que foi migrado para utf8.

Olhando as datas dos log do SVN e do lançamento desta versão 1.0.2, acredito
que essa versão já deveria estar com isso corrigido. :-/
Aqui no Ubuntu 10.10, versão do gpt 1.1 está tudo ok.

Então acho que o melhor é mesmo vc instalar essa versão mais nova a partir
do código-fonte, até pq tem outras coisas mais também.
Eu escrevi um tutorial para isso[0], mesmo que você nunca tenha feito esse
tipo de instalação, acredito que não terá dificuldades, mas se tiver é só
dizer.
