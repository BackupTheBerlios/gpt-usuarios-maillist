** Brief description of the package:

The project aims to create a programming environment in the portuguese
language. Version 1.0 includes a program which acts as a compiler,
translator and interpreter for a portuguese programming language known as
Portugol.

Virtually, every computing course in Brazil teaches some form of portuguese
pseudocode to teach programming to students. So, initially, this
project benefits those students.

Also, I understand that there are many portuguese-speakers who don't know
english, and I think the barrier to learn programming and program their
own computers should be minimized.

Programming is a very interesting activity that stimulates our minds.
Therefore, this project should offer a foundation to these people interested
in creating their own programs in their computers, even without understanding
english.
