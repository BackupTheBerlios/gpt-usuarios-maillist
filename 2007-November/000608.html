Oi pessoal.

Ainda faltam inúmeras melhorias no código do GptASM e GptVM, mas aos
poucos a gente vai evoluíndo :-)

Quanto as funcionalidades, o GptASM está assemblando quase todos os
códigos gerados pelo compilador. Falta endereçamento (ponteiros) e os
mnemônicos de matrizes, se eu não me engano. Claro, falta gerar um
report de erros descente, mas esse eu vou copiar do compilador quando
der tempo hehehe.

Quanto a GptVM, os opcodes referentes a inteiros estão ok. Saltos
condicionais ou não também estão funcionando. Chamada de procedimentos
da linguagem somente o "imprima" funciona, e ainda não é a versão
final (imprima será uma chamada a biblioteca de sistemas futuramente).
Como a VM não tem pilha de execução, ainda não temos suporte a
procedimentos definidos pelo usuário. Claro, também falta suporte aos
outros tipos de dados (string, matrix, real, ...)

Segue abaixo alguns exemplos em GptAssembly e a saída gerada pela GptVM:

***** EXEMPLO 1 *****

$ cat invocando_subrotinas_linguagem_1.gasm

program exemplo

proc main
    push_sp
    push "Olá mundo !!!"
    push_string
    push 1
    pcall imprima
    pop_sp

    exit 0
endproc

endprogram

$./gptvm invocando_subrotinas_linguagem_1

GptVM, release 0.1.0 (development release)
Olá mundo !!!

***** EXEMPLO 2 *****

$ cat variaveis_1.gasm

program exemplo

var x int
var y int

proc main
    var t1 int
    var t2 int

    // x := 10;
    iset x, 10

    // y := 20;
    iset y, 20

    // imprima( "x*y+4=", x*y+4 );
    push_sp
    imul t1, x, y
    isum t2, t1, 4
    push t2
    push_int
    push "x*y+4="
    push_string
    push 2
    pcall imprima
    pop_sp

    exit 0
endproc

endprogram

$ ./gptvm variaveis_1

GptVM, release 0.1.0 (development release)
x*y+4=204

***** EXEMPLO 4 *****

$ cat estruturas_condicionais_1.gasm

program exemplo

var x int

proc main
    var t1 int

    // x := 10;
    iset x, 10

    // se x >= 20 entao
    ige t1, x, 20
    ifnot t1, senao

    // x := 2;
    iset x, 2

    jmp proximo
    senao:

    // x := 1;
    iset x, 1

    proximo:

    // imprima( x );
    push_sp
    push x
    push_int
    push 1
    pcall imprima
    pop_sp

    exit 0
endproc

endprogram

$ ./gptvm estruturas_condicionais_1

GptVM, release 0.1.0 (development release)
1

    var t1 int
