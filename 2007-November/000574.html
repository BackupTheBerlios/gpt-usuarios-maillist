Pro gpt-editor ser compilado no (K)ubuntu, acho que vc vai precisar de
uma pancada de pacotes. Eis o que fiz pra instalar aqui no kubuntu:

$ sudo bash
# aptitude install g++ make xorg-dev libqt3-headers libqt3-mt-dev kdelibs-dev

-- algum tempo depois....

# cd gpteditor-1.0
# ./configure --prefix=/usr
# make
# export KDEDIR=/usr
# make install

Se não fizer o export do KDEDIR, o "make install" vai dar erro. Algo
como "cp: cannot create regular file
/share/apps/katepart/syntax/portugol.xml: No such file or directory".
