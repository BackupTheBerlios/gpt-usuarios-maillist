Bom, aproveitando o gás, acabei de fazer a integração dos subprojetos
(common, gptasm, gptc e gptvm, por enquanto).

Cada subdiretório tem um Makefile.boot, que instala os arquivos
Makefile.in, configure, etc específicos para o subprojeto. O diretório
raiz (gpt2) também tem esse arquivo, que faz o bootstrap do projeto
inteiro.

Para compilar tudo, basta fazer:

$ cd gpt2; make -f Makefile.boot
$ mkdir _build && cd _build
$ ../configure
$ make

Para fazer uma limpeza:
$ make Makefile.boot distclean

Para trabalhar em um subprojeto individualmente que dependa de
"common", é necessário construir a lib do subprojeto common antes de
qualquer coisa.
