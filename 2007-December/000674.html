>
>
>  Não exatamente.
>
> Para INT, CHAR e BOOL: sizeof(int);
>  Para REAL: sizeof(double);
>  Para STRING: sizeof(char)+sizeof(void*);
>  Para MATRIX: sizeof(void*);
>  Para DATA: tamanho definido pelo usuário.
