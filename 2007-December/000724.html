Vocês teriam alguma sugestão para melhorar o formato quanto ao tipo pointer
? De cara eu já diria que o "*" é desnecessário. Se é pointer, é óbvio que
vai ter o "*".

Ao invés de :

function fopen(string name, string mode) : pointer "FILE*" ::= fopen(name,
mode);

Eu pensei em algo como:

function fopen(string name, string mode) : pointer[FILE] ::= fopen(name,
mode);

Ou ainda:

function fopen(string name, string mode) : FILE ::= fopen(name, mode);

Sugestões ?
