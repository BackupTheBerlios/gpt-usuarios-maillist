Infelizmente, para instalar o editor no ubuntu é preciso instalar uma
quantidade considerável de bibliotecas antes. Eu não tenho o ubuntu
instalado aqui, então não sei qual seria o passo a passo para realizar
esta instalação neste sistema. Sei que você precisará instalar a
biblioteca do KDE3 (que eu não sei se o ubuntu está disponibilizando)
para desenvolvimento e o compilador GCC, no mínimo, para poder
compilar o GPTEditor. É uma pena :(

Mas, ainda assim, você pode usar o GPT, escrever algoritmos e
executá-los, sem precisar do GPTEditor.
