Acho que seria interessante utilizar o NSIS para a criação do instalador. Ele possui mais recursos e tem mais a ver pois é software livre.

Existem duas maneiras de se criar instaladores por ele:
*Programando linha por linha e...
*Através de alguns softwares que implementam interface gráfica para o NSIS

Página do Projeto:
http://nsis.sourceforge.net/Main_Page

Algumas Interfaces Gráficas:
http://sourceforge.net/projects/nsis-nice
http://www.psenica.com/nsis/
http://ultramodernui.sourceforge.net/
http://www.kaju74.de/index.php?nsisfd
http://nsis.sourceforge.net/NSIS_Quick_Setup_Script_Generator
http://nsis.sourceforge.net/Intelligent_NSIS_Script_Generator

Caso vocês queiram posso criar o instalador, já usei essas interfaces para
criar alguns instaladores :)
