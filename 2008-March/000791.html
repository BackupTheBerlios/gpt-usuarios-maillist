O arquivo que gera o setup é o seguinte:

http://svn.berlios.de/wsvn/gpt/trunk/gpt/packages/win_setup/setup.iss

Ele detalha todos os arquivos que são instalados, atalhos e mudanças
no registro do windows.

Mais fácil, porém é utilizar os arquivos instalados no diretório
c:/gpt pelo instalador atual. Além dos arquivos instalados note:

1) Os atalhos criados no menu iniciar;

2) As variáveis de ambientes criadas/alteradas.

Duas variáveis de ambiente são gravadas no registro do windows: PATH e
GPT_INCLUDE. Em PATH, é adicionado o caminho c:\gpt\bin e o
GPT_INCLUDE possui o valor "c:\gpt\lib\base.gpt" (claro, dependendo de
onde o gpt foi instalado).

Quanto ao notepad++, os arquivos se encontram em c:\gpt\notepad++. Se
o usuário escolhe fazer a integração, o instalador copia esses
arquivos para o diretório c:\Documents and Settings\<fulano>\Dados de
Aplicativos\Notepad++ (isso está documentado no
c:\gpt\notepad++\LEIAME.txt) se não me engano.
