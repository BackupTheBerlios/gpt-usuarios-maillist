Já está disponível a versão 1.0.1 do compilador GPT. Esta versão visa,
principalmente, corrigir bugs antigos e fornecer um ambiente mais
amigável em sistemas MS Windows por meio da integração com o editor
Notepad++ (http://notepad-plus.sourceforge.net/). Durante a instalação
do GPT, se o Notepad++ já se encontrar instalado, este último será
automaticamente adaptado para a linguagem G-Portugol.

Em breve será disponibilizado um tutorial explicando como usar o
Notepad++ para escrever programas com o GPT.
